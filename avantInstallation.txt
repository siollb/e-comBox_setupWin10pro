Cette exécutable ne doit être utilisé que sur Windows 10 professionnel, education ou entreprise. Un autre exécutable est prévu pour Windows 10 famille.

Vous vous apprêtez à installer l'application e-comBox qui nécessite les composants suivants :
 - HyperV
 - Docker Dekstop pour Windows 10
 - Git

Pour installer ces composants, vous devez disposer :
  - des droits d'administrateur ;
  - d'au moins 4 Go de RAM (12 Go minimum sont recommandés pour pouvoir lancer de nombreux sites) ;
  - d'au moins 20 Go de libre sur le disque dur (selon le nombre de sites installés, l'utilisation de l'espace disque sera variable, il ne pourra cependant pas excéder 64 GO).

Selon le débit de votre connexion Internet et la puissance de votre machine, l'installation sera plus ou moins longue.

Lorsque cela est demandé, vous devrez procéder au redémarrage de votre machine (probablement au nombre de deux). Fermez toutes les applications et enregistrez votre travail en cours !
