Cette exécutable ne doit être utilisé que sur Windows 10 professionnel, education ou entreprise. Un autre exécutable est prévu pour Windows 10 famille.

Vous vous apprêtez à installer l'application e-comBox qui nécessite les composants suivants :
 - HyperV
 - Docker
 - Git

Pour installer ces composants, vous devez disposer :
  - des droits d'administrateur ;
  - d'au moins 4 Go de RAM (12 Go minimum sont recommandés pour pouvoir lancer de nombreux sites) ;
  - d'au moins 80 Go de libre sur le disque dur.

Lorsque cela est demandé, vous devrez procéder au redémarrage de voutre machine (probalement au nombre de deux). Fermez toutes les applications et enregistrez votre travail en cours !
